# Calculator

Профессиональный калькулятор на Go с поддержкой точных вычислений.

# Возможности

- Поддержка операций: `+`, `-`, `*`, `/`
- Работа с десятичными числами (до 10 знаков после запятой)
- Поддержка как точки, так и запятой в качестве разделителя
- Автоматическое определение точности результата
- Два интерфейса: CLI и Web
- Graceful shutdown для веб-сервера
